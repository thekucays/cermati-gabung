# Cermati Gabung Page

## Stack
- python
- pytest-playwright


## How to Run 
- Install requirements using pip3
```
pip3 install -r requirements.txt
```

- Install playwright 
```
playwright install
```

- Run the test 
### headless mode
```
pytest test\gabung.py
```

### headed mode
```
pytest test\gabung.py --headed
```
